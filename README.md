# M&M Community Platform
Zentrales Backend für die M&M Community – gesteuert durch den **zahnlosen Frosch** (Gemini 2.0).

## Funktionen
* Kommunikation mit Gemini API
* Zentrale API-Schnittstelle für die Community
* Gehostet auf Render.com

## Setup
Das Projekt nutzt `hatchling` als Build-System und wird automatisch auf Render bereitgestellt.
